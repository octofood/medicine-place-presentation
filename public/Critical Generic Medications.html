The Problem

What a shortage looks like today

When an oral antibiotic supply chain breaks, here's how it plays out.

Week 1
Ottawa doesn't see it
  • API disruption stays offshore
  • Inventory can mask the problem
  • No real-time upstream view
Weeks 2-4
Pharmacies feel it
  • Wholesaler backorders appear
  • Pediatric supply gets reserved
  • Substitutions and callbacks begin
Weeks 4-8
Hospitals start rationing
  • Allocation and conservation rules
  • Emergency imports are pursued
  • Clinical workaround load rises
Months 3-12
Supply returns, unevenly
  • Foreign-authorized supply arrives
  • Labels, dosing, and counselling shift
  • Recovery is patchy and slow

Pharmacies feel it in weeks. Supply stabilizes in months. Nobody has a clear view while it's happening.

Appendix — G7 Compared

Phased Delivery

Start small. Expand after proof.

This is a first step on Canada's shortage-prone generics — not a plan to replace the whole market.

Phase 1

Start with a small set of oral CVDL medicines

  • Lower execution risk than a broad launch
  • Easier for officials to evaluate
  • Faster federal review on a narrow set
  • An opening dozen, not a locked list
Phase 2

Expand only after proof

  • Add more orals only after phase one proves out
  • Consider complex formats only after readiness
  • Tie expansion to demand, regulatory, and operating milestones
  • Scale only if the federal route actually works
